Miso Phone DB

Esta URL publica una portada informativa. La aplicacion completa no puede ejecutarse en GitHub Pages.

El proyecto usa Next.js con autenticacion, acciones de servidor, Prisma y PostgreSQL. GitHub Pages solo sirve archivos estaticos, asi que no puede levantar el login, la base de datos ni el backend de la app.

  • Frontend y backend: Next.js 16
  • Autenticacion: Auth.js con credenciales
  • Persistencia: Prisma + PostgreSQL
  • Despliegue recomendado: Railway para app y base de datos
Ver repositorio Ver instrucciones de despliegue

Si quieres una version publica funcional, despliegala en un servicio con runtime de Node.js y configura las variables de entorno como DATABASE_URL y AUTH_SECRET.
